# Sha API Backend



## Setup for dev
1. Install dependencies
   ```bash
   npm install
   ```

2. Configure `.env` file like `.env.dev` or `prod`

3. Generate prisma
   ```bash
   prisma generate
   ```

4. Start server
   ```bash
   npm run dev
   ```

## Creating or update model

 ```bash
   npx prisma migrate dev --name desc
```

 ```bash
   npx generate
```
